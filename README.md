== README

Soundflake was created as Project 2 for General Assembly's Web Development Immersive in September 2015.
Contributors: Robinson Lam, Phill Eltakchi, Alex Swan

The app uses the Soundcloud API 


* Technologies used
  - Ruby on Rails
  - Backbone
  - Soundwave API
  - Web Audio API
  - Paper.js
